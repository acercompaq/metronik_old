Máquina para tejido metálico MK-L100

Esta máquina está construída para producir la denominada lana o tejido metálico, con hilo metálico de

3,1 mm de diámetro. Tejido metálico en acero inox serie AISI 400, con hilo metálico de 3,1 mm de diámetro.

 

Corrugadora para tejidos MK-STR3000

Esta máquina es construída para conseguir el corrugado de tejidos varios.

Principalmente se usa en: tejidos de algodón, seda, poliester, etc, para cortinas y confecciones en general.

 

Etiquetadora BETA2000

Esta etiquetadora está construída para conseguir la transferencia de colores termoendurecidos sobre los tejidos. Se usa principalmente para confección de indumentaria de trabajo o deportiva.
